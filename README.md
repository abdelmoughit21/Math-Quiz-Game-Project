# 📌 Overview

This project is a complete console-based Math Quiz Game that generates random mathematical questions and evaluates the player's answers.

The project combines several important programming concepts into one practical application, making it an excellent learning project for beginner and intermediate C++ developers.

---

# 🚀 Features

✅ Randomly generated math questions  
✅ Multiple difficulty levels  
✅ Different operation types (+ − × ÷)  
✅ Input validation  
✅ Dynamic score tracking  
✅ Pass/Fail evaluation  
✅ Replay functionality  
✅ Structured game results  
✅ Console color feedback  
✅ Sound alerts for wrong answers
